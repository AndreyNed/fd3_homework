Экспорт в Excel на стороне клиента.

Файлы:

* ExcelConverter.js - компонент для осуществления экспорта.Принимает 2 параметра: данные для таблицы и флаг, который включает сам экспорт.

* excelTmpl.js - содержит функции-шаблоны и функцию-сборщик для создания XML-строки с данными таблицы. Сборщику можно передавать аргументы для наполнения и стилизации таблицы. Обязательным является массив ( 2-мерная матрица с текстовыми / цифровыми данными для самой таблицы, где массив с индексом [ 0 ] - заголовок, а все остальные - ячейки в строках с индексами, начиная с [ 1 ] ).

base64.js - содержит функции для кодирования / декодирования в base64 и utf-8.
